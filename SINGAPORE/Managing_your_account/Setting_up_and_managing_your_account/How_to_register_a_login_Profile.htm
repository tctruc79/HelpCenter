How to register a login Profile

Please follow these steps to register a login profile:

1. Go to web site https://transwap.com then click on button.

 

Or in the Login page, click on 'Register' link.

 

 

2. In 'Register your account screen', select type of account that you would like to create (Personal or Business).

 

 

3. To create Personal account, input required information:

  • Full name
  • Email
  • Password
  • Referral / promotion code (optional)

If all required fields are input, and the password is complicated enough (indicated by the password measurement), the  button will be enabled.

 

 

To create Business account, input required information:

  • Contact name
  • Business name
  • Email
  • Password
  • Referral / promotion code (optional)

If all required fields are input, and the password is complicated enough (indicated by the password measurement), the  button will be enabled.

 

 

4. After clicking on 'Register' button, the entered email will be checked whether it has been registered in any profile or not.

If yes, a duplicated email notification will be prompted.

Else, the ‘Verify your account’ screen will be taken place. At the same time, system will send a profile verification email to the registered email, with a ‘Verify’ link (or button).

 

5. If you got the verification email, you should click on the  button in the received email to proceed.

 

It comes to ‘Email verified’ screen, which has a link  that allows user to go back to ‘Sign in’ screen to login. How you are ready to sign in into system and follow next steps of Account Onboarding.

 

 

6. If no email sent / received for some reasons, please wait for 1 min then it is able to click the ‘Resend’ link again (this prevents our system from continuously re-sending email).

 

  • If less than 5 times of attempt, system will resend email once more time. Then back to step 5.
  • If exceeded 5 times of attempt, ‘Resend’ link will be changed to ‘Contact Support’ link, Our Support team is happy to help you at anytime. 

 

7. End of ‘How to register a login Profile’ Process.
